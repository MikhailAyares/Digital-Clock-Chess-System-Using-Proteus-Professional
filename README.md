# ♟️ Digital Chess Clock System (Pure Logic Design)

Welcome to the **Digital Chess Clock** project! This is a hardware-level simulation designed in **Proteus Professional** that relies entirely on digital electronics principles—no microcontrollers, no code, just pure sequential and combinational logic circuits at work. ⚡

## 📖 Overview
A fully functional digital chess clock system utilizing fundamental logic gates, 7-segment displays, NE555 timer ICs, and flip-flops. This project demonstrates the practical application of digital logic design to create a competitive dual-timer system with reliable state-switching capabilities.

## ✨ Key Features
* **Dual Countdown Timers:** Independent clock management for two players.
* **Hardware-Level State Switching:** Seamless player turn toggling using flip-flop logic circuits.
* **Clock Pulse Generation:** Accurate timing signals driven by the classic NE555 Astable Multivibrator.
* **Visual Feedback:** Direct hardware-to-display routing using 7-segment decoders.

## 🛠️ Components & Tech Stack
* **Simulation Environment:** Proteus Professional
* **Pulse Generator:** NE555 Timer IC
* **Memory & Control:** JK / D Flip-Flops
* **Logic Gates:** Standard TTL/CMOS ICs (AND, OR, NOT, NAND)
* **Outputs:** 7-Segment Displays & Decoders
